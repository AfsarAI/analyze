Welcome to the Data Processing Dashboard

This single-page application serves as a front-end for our automated data processing pipeline.Our backend script, execute.py, processes data from data.csv (originally data.xlsx)and generates statistical insights in result.json.

The results are automatically published to GitHub Pages via our CI/CD workflow.You can inspect the raw JSON output by clicking the "View Raw Results" link in the header.

How it Works

  • data.xlsx is converted to data.csv.
  • execute.py reads data.csv, performs calculations (e.g., sums, averages), and outputs result.json.
  • A GitHub Actions workflow (.github/workflows/ci.yml) lints the code with Ruff, runs execute.py, and publishes result.json to GitHub Pages.
  • This page serves as a simple front-end to explain the process and provide a direct link to the generated result.json.
